Deafy

Deafy - App de música para inclusão de deficientes auditivos

Um app de música com transcrição de áudio para textos/legendas feitas por inteligência artificial, postagem de música/podcasts ou audiobooks na home page para usuários verificados.

Front-end: Javascript, Typescript, NextJs, React Slider, Tailwindcss, React.

Back-end: Typescript, NestJs, MySQL, Prisma, Azure Cloud, JWT, API RESTful, MVC, HTTP.

Deploy: Vercel, Render.
